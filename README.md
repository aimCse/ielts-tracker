# ielts-tracker
Self-contained single-file web app to track your IELTS preparation across all 4 modules. Log every practice test, auto-calculate your band score using official Cambridge tables, identify weak areas, and monitor your progress toward your target band — all saved locally in your browser.

----------------------------------
IELTS Full Test Tracker — Band 8.5 Journey
A personal IELTS preparation tracker built as a single HTML file with zero dependencies. Designed for serious IELTS candidates targeting Band 7.5–9.
What it tracks:

Listening — Part 1–4 scores, auto band calculation, problem types & weak areas
Reading — Passage scores, accuracy %, time taken, weak question types
Writing — Task 1 & Task 2 bands, weighted average (T2 counts double), TA/TR · CC · LR · GR criteria tracking
Speaking — Part 1/2/3 bands, fluency, vocabulary, grammar & pronunciation issues

Features:

Auto band score calculation using official Cambridge score tables
Overview dashboard showing average band across all 4 modules and gap to target
All data saved in browser localStorage — no login, no server, no account needed
Fully offline — works by just opening the HTML file
Deployable anywhere — Vercel, GitHub Pages, Netlify, or just locally

Tech: Pure HTML · CSS · Vanilla JavaScript — no frameworks, no build tools.
